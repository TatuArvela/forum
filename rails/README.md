# Rails / Ruby

## Installation

Run the commands below and follow their instructions:

```bash
bundle install
yarn install
bin/rails db:migrate RAILS_ENV=development
rails server
```

Open [http://127.0.0.1:8001](http://127.0.0.1:8001)
