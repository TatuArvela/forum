# Rails / Ruby

## Installation

* Ruby >=2.7
* Bundler

### Instructions

1. Install the dependencies
    ```bash
    bundle install
    yarn install
    ```
2. Run the migrations
    ```bash
    bin/rails db:migrate RAILS_ENV=development
    ```
3. Run the server
    ```bash
    bin/rails server
    ```

4. Navigate to [http://127.0.0.1:8001/](http://127.0.0.1:8001/) in your browser
